# 每日早报自动化系统 - MVP版本设计

## 🎯 系统目标
创建一个每日早上8:00推送的综合性早报，包含：
1. **科技新闻** (基于tech-news-digest)
2. **天气预报** (主要城市)
3. **简单财经** (股票指数、加密货币)
4. **重要新闻** (精选要闻)

## 📁 系统结构
```
daily-briefing/
├── config/
│   ├── sources.json           # 数据源配置
│   ├── cities.json            # 天气预报城市
│   └── templates/
│       ├── discord.md         # Discord模板
│       └── markdown.md        # Markdown模板
├── scripts/
│   ├── collect-weather.py     # 收集天气数据
│   ├── collect-finance.py     # 收集财经数据
│   ├── collect-news.py        # 收集新闻要闻
│   └── generate-briefing.py   # 生成早报
└── archive/                   # 早报存档
```

## 🔧 技术实现

### 1. 天气预报模块
使用wttr.in API，支持中国主要城市：
- 北京、上海、广州、深圳、杭州、成都、武汉、西安

### 2. 财经数据模块
使用公开API获取：
- A股指数 (上证、深证)
- 美股指数 (道琼斯、纳斯达克)
- 加密货币 (BTC、ETH价格)

### 3. 新闻要闻模块
使用RSS订阅：
- 新华网要闻
- 人民网时政
- 央视新闻

### 4. 科技新闻模块
直接复用tech-news-digest技能
